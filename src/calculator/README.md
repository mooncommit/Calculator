# 🕶️ Java Calculator

STEP2 까지 완성한 사칙연산 계산기

---

## 프로젝트 소개


두 개의 양의 정수를 입력받고 사칙연산 결과를 출력해줍니다.  
계산 결과는 리스트에 저장되고, 조회하거나 삭제 기능이 있습니다.

---

## 기능

- 두 양의 정수 입력
- `+` `-` `*` `/` 사칙연산 입력
- 잘못된 입력 예외 처리 (음수, 0으로 나누기, 잘못된 연산자)
- 계산 결과 저장 및 조회
- 첫 번째 결과 삭제 기능
- exit 입력 시 프로그램 종료
